# Kanji Table 2136

일본어 상용한자표 테이블 프로젝트입니다.

## Features

- 총 2136개 한자
  - 좌우 방향키(←/→) 또는 이전/다음 버튼으로 페이지 이동
  - 버튼 그룹
    - `고정` 버튼(한/일/예)으로 표시 모드 토글
    - `홀드` 버튼(한/일/예)을 누르는 동안만 표시
  - 각 버튼의 기능
    - `한`: 한국어 훈/음 표시
    - `일`: 일본어 뜻/음(훈독/음독) 표시
    - `예`: 일본어 단어 예시 1~2개 표시
- 암기 표시 (셀 배경색)
  - 암기 상태 보존(`localStorage`)
  - 단일셀 토글: 셀 클릭
  - 다중 선택
    - `PATH`: 드래그 커서 경로
    - `RECT`: 사각 영역 선택

# Kanji Table 2136

A vanilla JavaScript project for browsing Joyo Kanji in a table layout.

## Features

- 2,136 kanji
  - Page navigation with Left/Right arrow keys (←/→) or 이전(Prev)/다음(Next) buttons
  - Button groups
    - `고정(Pin)` buttons (`한`/`일`/`예`) to toggle display modes
    - `홀드(Hold)` buttons (`한`/`일`/`예`) to show content only while pressed
  - Function of each button
    - `한(ko)`: Korean meaning/reading
    - `일(ja)`: Japanese meaning/readings (kun/on)
    - `예(ex)`: 1-2 Japanese word examples
- Memorization state (cell background color)
  - Memorization state persistence (`localStorage`)
  - Single-cell toggle: click a cell
  - Multi-selection
    - `PATH`: drag cursor path
    - `RECT`: rectangular area selection

# Kanji Table 2136

常用漢字テーブルを閲覧するための Vanilla JavaScript プロジェクトです。

## Features

- 合計 2136 字
  - 左右キー(←/→)または이전(前)/다음(次)ボタンでページ移動
  - ボタングループ
    - `고정(固定)`ボタン(한/일/예)で表示モードを切り替え
    - `홀드(ホールド)`ボタン(한/일/예)を押している間だけ表示
  - 各ボタンの機能
    - `한(韓)`: 韓国語の訓/音を表示
    - `일(日)`: 日本語の意味/読み(訓読/音読)を表示
    - `예(例)`: 日本語の単語例を1-2個表示
- 暗記状態の表示（セル背景色）
  - 暗記状態を保持（`localStorage`）
  - 単一セルの切り替え: セルをクリック
  - 複数選択
    - `PATH`: ドラッグカーソルの経路
    - `RECT`: 矩形領域選択
